# Realtime Firebase Demo

This document provides instructions on how to create a new Git repository and push the Realtime Firebase project to GitHub using the Git command line interface.

## Steps to Create a Git Repository and Push to GitHub

1. Navigate to your project directory:
   ```
   cd path/to/realtime-firebase-demo
   ```

2. Initialize a new Git repository:
   ```
   git init
   ```

3. Add all files to the staging area:
   ```
   git add .
   ```

4. Commit the changes with a message:
   ```
   git commit -m "Initial commit"
   ```

5. Create a new repository on GitHub (do this through the GitHub website).

6. Add the remote repository URL (replace `<URL-do-repositório>` with your GitHub repository URL):
   ```
   git remote add origin <URL-do-repositório>
   ```

7. Push the changes to the GitHub repository:
   ```
   git push -u origin master
   ```

These commands will set up your Git repository and push your project to GitHub.
